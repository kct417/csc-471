# CSC 471 Lab 3

## Getting Started

```
git clone --recursive -j4 https://github.com/kct417/csc-471-lab3.git
```

## Build

```
./build.sh
```

**NOTE:** This builds with `CMake` behind the scenes

### Build with `CMake`

```
mkdir build
cd build
cmake ..
make
```

## Run

```
./run.sh
```

**NOTE:** This builds and runs from the root directory

### Run from build directory

```
./lab3
```
